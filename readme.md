# Wallapop Clone

Wallapop Clone es un proyecto desarrollado con JavaScript utilizando la arquitectura MVC.  
Este proyecto fue realizado como parte del Bootcamp de KeepCoding.

La aplicación permite:

- Registro e inicio de sesión de usuarios
- Creación de productos
- Edición de productos
- Visualización del detalle de productos
- Filtrado dinámico de productos
- Paginación de resultados

---

# Tecnologías

- JavaScript
- HTML5
- CSS3
- Arquitectura MVC
- JSON-Server
- Fetch API

---

# Instalación del proyecto

## 1. Clonar el repositorio

```text
git clone <url-del-repositorio>
```

---

## 2. Configuración del servidor JSON-Server

El proyecto utiliza un servidor externo con JSON-Server para simular la API REST.

El archivo `db.json` incluido en este repositorio contiene toda la estructura y data previamente configurada, por lo que deberá ser copiado o movido al proyecto donde se ejecutará JSON-Server.

El servidor deberá estar configurado en el puerto:

```text
http://localhost:8000
```
